Right now I'm working on a project called LIRO. It's a video and photo hosting site with a blog section as well. It's been a passion project since Vimeo was going to begin charging $300/month to host videos. Right now it is live on a Raspberry PI in my house. I've connected it to a domain by getting a static IP address and routing it through my router to the PI. It's currently built using React, FastAPI, SQLite, and NGINX for reverse proxy.
